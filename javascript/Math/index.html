Estudando Math

Tipos de Maths e suas funções
Nome Função
Math.round(x) Retorna o valor arredondado de x
Math.abs(x) Retorna o valor positivo de x
Math.sqrt(x) Retorna a raiz quadrada de x
Math.ceil(x) Retorna o valor de x arredondado para cima
Math.floor(x) Retorna o valor de x arredondado para baixo
Math.cos(x) Retorna o cosseno (um valor entre -1 e 1) do ângulo x (dado em radianos)
Math.sin(x) Retorna o seno (um valor entre -1 e 1) do ângulo x (dado em radianos)
Math.PI Retorna a razão entre a circuferência de um círculo e seu diâmetro
Math.random() Retorna um número aleatório entre 0 e 1
Math.pow(x, y) Retorna o valor de x á portência de y
Math.min(56, 3, -97, ,-2, 0, 17) Pode ser utilizado para encontrar o valor mais baixo, ou seja, -97
Math.max(56, 3, -97, ,-2, 0, 17) Pode ser utilizado para encontrar o valor mais alto, ou seja, 17
